# Введение в документацию TaskMate

Добро пожаловать в документацию TaskMate — системы управления задачами для автодилеров.

## О проекте

TaskMate — это комплексное решение для автоматизации рабочих процессов в автодилерах. Система включает веб-интерфейс для менеджеров, мобильный интерфейс для сотрудников и Telegram бота для уведомлений.

### Архитектура

- **Монолитное приложение** с модульной структурой
- **Многоарендность** (multi-tenant) для нескольких автодилеров
- **Гибридный подход** к работе с датами (UTC + timezone)
- **Асинхронная обработка** через RabbitMQ
